Utilizaremos o pino expansor P9.14 para o controle do nosso pwm do buzzer que é o EHRPWM1A,
equivalnete a: 
PWMSS1 0x4830_0000

Ativar o PCRM para PWMSS1 - FINALIZADO

Inicialização ------------------------------------------------------
Offsets: 
SYSCONFIG (0x04) - Configuraçoes gerais do pwm - FINALIZADO
STANDBYMODE = 1h (No-standby)
IDLEMODE = 1h (No-idle)
FREEEMU = 1h (Free Emulation)
BIN =  010110

CLKCONFIG (0x08) - interuptor para ligarmos e desligarmos o pwm - FINALIZADO
Utilizaremos o modulo ePWMCLK_EN para gerar as ondas

CLkSTATUS (0X0C) - Confirmar que o PCRM esta funcionando - FINALIZADO

Utilizando PWM -------------------------------------------------------
O PWM funciona com base em dois contadores, internamente existe um clock e um contador o teto do contador ate onde
vai contar é o TBPRD 0 ate o TBPRD quem dara o ritimo e ficara monitorando esse contador é o CMPA uma relação de 50%
entre esses dois significa que nesse contador ele estara observando o contador quando ele estiver abaixo de 50% manda
sinal 1 acima de 50% do contador TBPRD zero o TBRD atinge seu pico reinicia a contagem 

Para podermos utilizar precisamos de duas informaçoes:
TBPRD: Que nos dara o Teto do contador
